# 🐳 Project 2: Docker Basics - Your First Container

## 👋 Welcome Beginner

This project will teach you Docker from absolute scratch. No prior knowledge needed!

## 🎯 What You'll Learn

1. What is Docker and why we use it
2. How to create your first container
3. How to build Docker images
4. How to run applications in containers

## ⏱️ Time Required

30-45 minutes

## 📋 Prerequisites

- A computer (Mac, Windows, or Linux)
- Internet connection
- That's it! We'll install everything together

## 🚀 Quick Start

1. Read: [`01-WHAT-IS-DOCKER.md`](01-WHAT-IS-DOCKER.md) - Understand Docker in simple terms
2. Follow: [`02-INSTALLATION.md`](02-INSTALLATION.md) - Install Docker on your computer
3. Practice: [`03-FIRST-CONTAINER.md`](03-FIRST-CONTAINER.md) - Run your first container
4. Build: [`04-BUILD-YOUR-IMAGE.md`](04-BUILD-YOUR-IMAGE.md) - Create your own Docker image
5. Code: [`05-SIMPLE-APP.md`](05-SIMPLE-APP.md) - Containerize a simple web app
6. Learn: [`06-DOCKER-COMMANDS.md`](06-DOCKER-COMMANDS.md) - Essential Docker commands

## 📚 Learning Path

```
Start Here → What is Docker? → Install Docker → Run Container → Build Image → Deploy App
```

## 🎓 After This Project

You'll be able to:

- ✅ Explain what Docker is to anyone
- ✅ Run any Docker container
- ✅ Create your own Docker images
- ✅ Containerize simple applications
- ✅ Use Docker in interviews confidently
