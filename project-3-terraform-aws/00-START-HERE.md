# 🏗️ Project 3: Terraform + AWS Infrastructure

## 👋 Welcome Beginner

This project will teach you **Infrastructure as Code (IaC)** using Terraform and AWS from absolute scratch. No prior cloud or Terraform experience needed!

## 🎯 What You'll Learn

1. What is Infrastructure as Code (IaC)
2. What is Terraform and why we use it
3. AWS basics (EC2, S3, VPC)
4. How to write Terraform code
5. How to deploy real infrastructure
6. Best practices and state management

## ⏱️ Time Required

6-8 hours (including AWS setup)

## 📋 Prerequisites

- A computer (Mac, Windows, or Linux)
- Internet connection
- AWS account (free tier is enough)
- Credit/debit card for AWS (won't be charged if you stay in free tier)
- Basic command line knowledge
